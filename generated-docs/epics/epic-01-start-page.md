# Epic 1: Start Page & Report Batch Management

## Description
Provide the main entry point for users to view, search, and manage monthly report batches. Users can see all existing batches, create new batches, and navigate to batch details or logs. This is the first screen users see when they access the application.

## Wireframe Reference
- [Screen 1: Start Page (Batches List)](../wireframes/screen-01-start-page.md)

## Business Value
- Central hub for all monthly report batch operations
- Quick access to batch status and navigation
- Foundation for entire monthly workflow

## Stories

1. **View Report Batches List** - Display existing monthly report batches with status, search, and pagination
   - File: `story-01-01-view-report-batches.md`
   - Status: Pending

2. **Create New Report Batch** - Form to create new monthly batch with month/year selection and optional SFTP auto-import
   - File: `story-01-02-create-batch.md`
   - Status: Pending

3. **Navigate to Batch Details** - View Details button navigates to Monthly Process Workflow View
   - File: `story-01-03-navigate-batch-details.md`
   - Status: Pending

4. **Navigate to Batch Logs** - View Logs button navigates to Monthly Process Logs screen
   - File: `story-01-04-navigate-batch-logs.md`
   - Status: Pending

5. **Export Batch List** - Export report batches to CSV
   - File: `story-01-05-export-batch-list.md`
   - Status: Pending

## Dependencies
None - This is the foundation epic

## Acceptance Criteria (Epic-Level)
- [ ] Users can view all report batches in a searchable, paginated table
- [ ] Users can create new batches with valid month/year selection
- [ ] Users can navigate to batch workflow and log screens
- [ ] Users can export batch data for external analysis
