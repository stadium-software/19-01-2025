# Step 5: Troubleshooting & Support

If you encounter any issues during onboarding or development, use this guide to resolve common problems and find help.

- **Common Issues:**
  - Dependency installation errors
  - Environment variable setup problems
  - Quality gate failures (lint, type check, tests)
  - Agent workflow interruptions

- **Solutions:**
  - Check the getting started checklist for step-by-step verification
  - Review error messages and follow suggested fixes
  - Restart your development server or re-run setup commands

- **Get Help:**
  - Consult the documentation links in the README
  - Reach out to your team or project maintainer for support

**What's next?**
Once resolved, continue with your workflow or getting started steps.
